# Sistema de Gestión Hotelera

## Hotel Sabana Brava 314

Sistema de información orientado a la administración de reservas, habitaciones, empresas y facturación para el Hotel Sabana Brava 314.

</div>

---

## Descripción

Este proyecto busca automatizar y centralizar los procesos administrativos del hotel, permitiendo una mejor organización de la información y optimización de las operaciones diarias.

El sistema está orientado principalmente al manejo de hospedaje para empresas del sector petrolero y sus trabajadores.

---

## Funcionalidades principales

- Gestión de reservas
- Consulta de disponibilidad
- Asignación de habitaciones
- Gestión de habitaciones
- Registro de empresas
- Administración de contratos
- Facturación electrónica
- Consulta de facturación
